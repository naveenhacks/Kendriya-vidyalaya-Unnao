# KVISION - Kendriya Vidyalaya Unnao Platform

This is the refactored project structure for the KVISION platform.

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Configuration**
    Ensure environment variables for Supabase and Gemini AI are set in your Vercel project settings or `.env` file.

3.  **Run the Development Server**
    ```bash
    npm run dev
    ```
